# MyET – AI-Native News Experience
 Overview:-
MyET is an AI-powered platform that transforms traditional news into a personalized and interactive experience.
It leverages Generative AI to convert static news into context-aware insights, summaries, and conversations.

 Features:-
 * Personalized News Feed (based on user profile)
 * AI-powered News Chat (ask questions on news)
 * Smart Summarization (long → short insights)

 Architecture:-
The system uses a multi-agent architecture:
* User Agent
* News Ingestion Agent
* LLM Processing Agent
* Recommendation Agent

Tech Stack:-
* Python
* Streamlit
* LLM API (OpenAI / similar)
* News dataset / APIs

Setup Instructions:-
git clone https://github.com/ananyaacodes/MyET-AI-News.git
cd myet-ai-news
pip install -r requirements.txt
python app.py

Future Scope:-
* Real-time news APIs
* Multilingual support
* Advanced personalization
